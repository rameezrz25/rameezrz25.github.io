# Portfolio App

A modern, responsive developer portfolio built with React and Vite.

## Features

- **Modern UI**: Glassmorphism design with dark/light/sepia themes.
- **Responsive**: Fully optimized for mobile and desktop.
- **Animations**: Smooth transitions using Framer Motion.
- **Deployment**: Configured for automated deployment to GitHub Pages.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Deployment

The project is configured to deploy to GitHub Pages automatically via GitHub Actions.

1.  Push changes to the `master` branch.
2.  The workflow will automatically build and deploy the site.
3.  Ensure your GitHub repository settings have Pages enabled with Source set to **GitHub Actions**.
